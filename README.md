請每週必須捐贈一張卡牌，每週五會發佈踢人訊息，並且在每週日12:00之後開始進行結算，請多多注意。

如有原因而不能捐贈請在社團、部落或群組告知，我們會在將你排除於踢人名單，如果不小心被踢出部落，只要在盃數夠或者利用連結歡迎在回來。

踢人機制即將啟動

抱歉，因為你尚未捐贈卡牌，因此我們決定將你踢出部落。但只要盃數條件夠，歡迎隨時回來。
